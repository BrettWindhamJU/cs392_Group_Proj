# StockWise-Inventory Managment System 

This program is used to manage inventory systems for clients

## Description

Our Project is a inventory managment system mainly to be used by small businesses, it allows owners of buisnesses to set up and manage inventory data with the program, they can also assign employees as managers in order to have more people have 
the ability to manage the inventory of that location. The Owner also has the ability to have multiple locations with seperate inventories be managed by location specific employees.
